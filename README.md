# 🌧️ 초단기 참견예보

> 장마철 기상청 **초단기 강수예측(레이더)** 화면을 들여다보는 인턴 옆에서,
> 옆자리 4년차 사수 **조 매니저**가 훈수를 두는 인터랙티브 밈 웹.
> *"무릎이 슈퍼컴을 이긴다."*

React + Vite + Tailwind CSS v4 · 외부 API/에셋 없이 전부 자체 완결(빗소리까지 WebAudio 합성).

> ⚠️ **패러디입니다.** 실제 기상청/날씨누리 브랜드·로고를 사용하지 않으며, 강수 데이터는 전부 연출입니다. 진짜 예보는 [기상청 날씨누리](https://www.weather.go.kr).

---

## 실행

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 프로덕션 빌드 → dist/
npm run preview  # 빌드 결과 미리보기
```

## 화면 구성

- **왼쪽 — 인턴의 모니터**: 기상청 초단기예측 스타일 UI
  - `<canvas>` 한반도 레이더 + 서→동으로 이동·발달하는 강수 에코(강수강도 컬러스케일)
  - 카테고리 탭 · 권역 선택 · 발표시각 · 강수강도 컬러바(mm/h)
  - 관측/예측 구간이 나뉜 타임라인 슬라이더 + 재생/정지, 프레임 배지 `[예측 +40분]`
- **오른쪽 — 옆자리 조 매니저**: 데드팬 표정(눈썹 각도로만 감정 표현)의 인라인 SVG 캐릭터 + 노란 포스트잇 말풍선. 참견할 땐 의자를 굴려 다가옵니다.
- **하단 — 조작 패널**: 참견 게이지 · 강수 강도 슬라이더 · 액션 버튼

## 상호작용 (직장인 공감 개그)

| 트리거 | 조 매니저 반응 |
|---|---|
| **⟳ 새로고침 연타** | 참견 게이지가 차오르며 대사 폭주 → 만렙(8회)에 **우산을 쓱 내밀며** "이거 갖고 가. 내일 커피는 네가 사." |
| **🦵 무릎 예보** | 무릎이 빨갛게 욱신 → "비 와. 확정이야." 카드 + `무릎 적중률 100% vs 기상청 87%` |
| **🌂/☂️ 우산 챙겼어요?** | 챙기면 강수 셀이 회사를 슬쩍 비껴감(+ 평생 1회 칭찬 "오, 웬일로."), 안 챙기면 실눈 잔소리 |
| **🚨 부장님 온다! (`Space`)** | 레이더가 **가짜 엑셀 손익시트로 순간 위장** → "고개 숙여. 마우스 움직여." (3초 뒤 자동 복귀) |
| **🌧️ 코지 모드** | 저녁 무드 + 화면 빗줄기 + **합성 빗소리** |
| **가만히 응시 / 레이더 클릭** | 아이들 참견, 구름 콕 찌르기 이스터에그 |
| **강수 강도 슬라이더** | 맑음→호우 전환마다 자동 코멘트 (rainStart / heavyRain / clear) |

이스터에그: 파티션 명패의 *"옆옆자리: 나기상 대리"* 👀

## 구조

```
src/
├─ App.jsx                  # 상태 · 대사 트리거 오케스트레이션
├─ data/
│  ├─ dialogue.js           # 조 매니저 대사 뱅크(카테고리별) + 에스컬레이션
│  └─ radar.js              # 강수강도 스케일 · 권역 · 타임스텝 · 한반도 외곽선
├─ lib/rainSound.js         # WebAudio 빗소리 합성
└─ components/
   ├─ RadarCanvas.jsx       # 레이더 캔버스 애니메이션
   ├─ RadarPanel.jsx        # 기상청 스타일 UI 크롬 + 타임라인 + 컬러바
   ├─ MentorCharacter.jsx   # 조 매니저 SVG
   ├─ SpeechPostit.jsx      # 포스트잇 말풍선
   ├─ DisguiseSheet.jsx     # 부장님 대비 가짜 엑셀
   ├─ OfficePanel.jsx       # 사무실/파티션 + 캐릭터
   └─ ControlDock.jsx       # 참견 게이지 · 강도 · 액션 버튼
```
